# 💰 Personal Expense Tracker in C

A simple console-based Personal Expense Tracker developed in C programming language.  
This project helps users manage and track their daily expenses efficiently using file handling.

## 📌 Project Overview

The Personal Expense Tracker allows users to record, view and manage their expenses in a structured way.  
It is designed to simulate a real-world financial tracking system using core C programming concepts such as file handling, loops, functions and conditional statements.

## ✨ Features

- Add new expense records
- View all stored expenses
- Delete specific expense entries
- Generate monthly expense reports
- Category-wise expense tracking
- Budget monitoring based on monthly salary
- Persistent data storage using file handling

## 🛠️ Technologies Used

- C Programming Language
- File Handling in C
- Standard Input/Output Libraries

## 🎯 Learning Outcomes

Through this project, I gained hands-on experience in:

- Working with file handling in C
- Structuring a real-world console application
- Implementing CRUD operations (Create, Read, Update, Delete)
- Improving logical thinking and problem-solving skills
- Understanding how data persistence works without database
